# Credit Card Validator

## Описание

Консольное приложение на Go для проверки кредитных карт.
Программа проверяет формат номера, валидирует его по алгоритму Луна и определяет банк по BIN.

## Возможности

* Проверка формата номера карты (13–19 цифр)
* Проверка номера по алгоритму Луна
* Определение банка по первым 6 цифрам (BIN)
* Работа с данными из файла banks.txt

## Установка и запуск

1. Убедитесь, что у вас установлен Go
2. Склонируйте репозиторий:

```
git clone https://github.com/your-username/your-repo.git
cd your-repo
```

3. Создайте файл banks.txt в корне проекта:

```
Lunar Bank,400000,499999
Mars Credit Union,500000,599999
Venus Express Bank,600000,699999
Saturn Ring Financial,700000,799999
Jupiter Trust,800000,899999
```

4. Запустите программу:

```
go run main.go
```
